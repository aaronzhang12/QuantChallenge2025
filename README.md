# QuantChallengePlayground

This workspace contains a notebook to train baseline models on generated anonymous financial data.

Files:
- `data/train.csv` — training data (anonymous financial dataset).
- `train_model.ipynb` — notebook that performs EDA, preprocessing, trains baseline models, evaluates them, and saves the best model.
- `models/` — output directory where `best_model.joblib` will be saved after running the notebook.

Quick start

1. Create a Python environment and install dependencies (example):

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
